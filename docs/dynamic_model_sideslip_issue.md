# 动力学模型侧偏角导致横向跟踪误差持续增长

## 问题现象

使用动力学模型（`--plant dynamic`）跑圆弧轨迹时，横向误差随时间持续增长（RMSE ≈ 1.5m），
远大于运动学模型（RMSE ≈ 0.05m）。车辆始终偏向圆弧内侧，实际曲率大于参考曲率。

## 根因：侧偏角使航向 ≠ 速度方向

### 运动学 vs 动力学

| | 运动学模型 | 动力学模型 |
|---|---|---|
| 侧向速度 vy | 恒为 0（假设无侧滑） | 非零（轮胎力学决定） |
| 车辆航向 yaw | = 速度方向 | ≠ 速度方向 |
| 侧偏角 β = atan2(vy, vx) | 0 | ≈ 5°（R=30m, v=5m/s） |

### 侧偏角的物理来源

1. 圆弧行驶需要**向心力** F = m·v²/R
2. 向心力由**轮胎侧向力**提供：F_y = -C_α · α（侧偏刚度 × 轮胎侧偏角）
3. 轮胎要产生侧向力，其运动方向必须与轮胎朝向有夹角 → **轮胎侧偏角 α ≠ 0**
4. 后胎侧偏角 → 后轴横向速度 vy_r ≠ 0 → 传递到前轴 vy_f ≠ 0
5. 前轴 vy_f ≠ 0 → 车体侧偏角 β = atan2(vy_f, vx_f) ≈ 5°

**关键公式**（`dynamic_vehicle.py` 第 63-64 行）：
```python
alpha_f = atan2(vy_cg + lf*r, vx) - delta   # 前胎侧偏角
alpha_r = atan2(vy_cg - lr*r, vx)             # 后胎侧偏角
```

### 稳态转弯时侧偏角是常数

匀速圆弧 → yaw rate r = v/R 恒定 → 所需向心力恒定 → 轮胎侧偏角恒定 → vy 恒定 → β 恒定。

## 误差传播链

### 1. 侧偏角引入"假"航向误差

控制器计算航向误差时用的是**车辆航向 yaw**，而非速度方向：

```
heading_error = yaw - ref_theta
             = (velocity_direction - β) - ref_theta
             ≈ ref_theta - 5° - ref_theta     （当速度方向≈参考切线时）
             = -5°
```

即使车辆速度方向完美跟踪参考轨迹，控制器仍看到 **-5° 的假航向误差**。

### 2. 假航向误差驱动反馈往内转向

控制器根据 heading_error 计算 target_theta（期望航向修正），再转为反馈曲率。
-5° 的假航向误差使控制器认为"车头指向圆弧外侧"，于是**往内侧转向**。

### 3. 横向修正被 clamp 限制

横向误差（车在内侧）产生的 target_theta 修正方向是往外侧转，但被 T1 表限幅到 ±3.86°。
而假航向误差贡献约 -4°（经过系数折算），两者几乎抵消甚至反号。

### 4. 净效果：持续的曲率偏差

实测数据（t=8s, R=30m 圆弧, v=5m/s, 动力学模型）：

| 量 | 值 | 说明 |
|---|---|---|
| sideslip | +5.26° | β = atan2(vy_f, vx_f) |
| heading_error | -3.89° | yaw - ref_theta，控制器认为车头偏外 |
| lateral_error | +1.26m | 车在圆弧内侧 |
| steer_fb | +4.1° | **正值 = 往内侧转**，尽管车已在内侧 |

反馈在往内侧转（+4.1°），而车已经在内侧（+1.26m）——这就是误差持续增长的原因。

### 5. 对比运动学模型

运动学模型 β = 0，heading_error 纯粹反映真实航向偏差，反馈方向始终正确，
误差快速收敛到稳态（RMSE ≈ 0.05m）。

## 本质总结

```
侧偏角 β ≈ 5°（常数）
→ 假航向误差 ≈ -β（常数偏差）
→ 恒定的曲率偏差（反馈往内）
→ 等效于航向通道的"斜坡扰动"（ramp disturbance）
→ P 控制器无法消除斜坡扰动
→ 横向误差持续增长（被 P 反馈部分抑制为近似线性增长）
```

## 可能的修复方向（待实验）

1. **控制器用速度方向替代航向**：heading_error = atan2(vy, vx) 旋转后 - ref_theta，消除假航向误差
2. **侧偏角补偿**：估算 β 并从 heading_error 中减去
3. **增大 T1 限幅**：让横向修正不被假航向误差吃掉
4. **添加积分环节**：消除稳态偏差（但需注意积分饱和和稳定性）
5. **训练自适应**：让优化器通过调整 T3（reach_time_theta）等参数间接补偿

## 影响范围

- 仅影响**动力学模型**（运动学模型 β=0 无此问题）
- 仅在**有持续曲率的轨迹**上显著（圆弧最严重，正弦次之，直线无影响）
- 当前可微调参在动力学模型上的效果被此问题限制
